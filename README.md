# erie21
ERIE21 Tech Day

Exercises to have fun with Raspberry Pi and SenseHat
https://projects.raspberrypi.org/en/projects/getting-started-with-the-sense-hat/3


## Installation

Clone the repository: `git clone git@github.com:srizzo-metis/erie21.git`

Install requirements
```bash
  pip3 install -r requirements.txt
```
Run the programs
`python3 ex1.py`
`python3 ex2.py`
`python3 ex3.py`
`python3 ex4.py`
`python3 ex5.py`
`python3 ex6.py`
